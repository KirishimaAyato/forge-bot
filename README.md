# 🚀 FORGE BOT — Инструкция по деплою

## Структура проекта
```
forge_bot/
├── main.py          # Точка входа (запускает бот + API)
├── bot.py           # Telegram бот
├── api.py           # FastAPI сервер (раздаёт HTML + принимает данные)
├── database.py      # SQLite база данных
├── requirements.txt # Зависимости Python
├── railway.toml     # Конфиг Railway
├── .env.example     # Пример переменных окружения
└── webapp/
    └── index.html   # Твоё приложение FORGE
```

---

## Шаг 1 — Создай бота в Telegram

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя (например: `FORGE Workout`)
4. Придумай username (например: `forge_workout_bot`)
5. BotFather даст тебе **токен** — сохрани его, он нужен на шаге 3

---

## Шаг 2 — Загрузи код на GitHub

1. Зайди на **github.com**, создай аккаунт если нет
2. Нажми **"New repository"**, назови `forge-bot`
3. Загрузи все файлы из папки `forge_bot/`:
   - Либо через сайт GitHub (перетащи файлы)
   - Либо через терминал:
     ```bash
     cd forge_bot
     git init
     git add .
     git commit -m "Initial commit"
     git remote add origin https://github.com/ТВОЙ_ЮЗЕРНЕЙМ/forge-bot.git
     git push -u origin main
     ```

---

## Шаг 3 — Деплой на Railway

1. Зайди на **railway.app** → "Start a New Project"
2. Выбери **"Deploy from GitHub repo"**
3. Подключи свой аккаунт GitHub, выбери репозиторий `forge-bot`
4. Railway начнёт сборку — подожди 1-2 минуты

### Добавь переменные окружения в Railway:
Зайди в проект → вкладка **Variables** → добавь:

| Переменная | Значение |
|-----------|---------|
| `BOT_TOKEN` | Токен от BotFather |
| `WEBAPP_URL` | Пока пусто — заполним ниже |
| `DB_PATH` | `/data/forge.db` |

5. Зайди в **Settings → Domains** → нажми **"Generate Domain"**
6. Railway даст тебе URL вида `https://forge-bot-production.up.railway.app`
7. Скопируй этот URL и добавь его в переменную `WEBAPP_URL`
8. **Redeploy** (нажми Deploy снова)

---

## Шаг 4 — Настрой кнопку в BotFather

1. Напиши BotFather: `/setmenubutton`
2. Выбери своего бота
3. Отправь URL своего приложения: `https://forge-bot-production.up.railway.app`
4. Название кнопки: `⚔ FORGE`

---

## Шаг 5 — Проверь

1. Найди своего бота в Telegram
2. Напиши `/start`
3. Нажми кнопку **"⚔ Открыть FORGE"**
4. Приложение откроется прямо внутри Telegram!

## Команды бота

| Команда | Что делает |
|---------|-----------|
| `/start` | Главное меню + кнопка открытия |
| `/stats` | Статистика в виде текста |
| `/water` | Сколько воды выпито сегодня |
| `/reset` | Сбросить весь прогресс |

## Как работает синхронизация данных

```
Телефон/ПК                    Сервер Railway              База данных
    │                               │                           │
    │── открываю FORGE ────────────>│                           │
    │                               │── load state ────────────>│
    │                               │<─ данные ─────────────────│
    │<─ показывает мой прогресс ────│                           │
    │                               │                           │
    │── записываю тренировку ──────>│                           │
    │                               │── save state ────────────>│
    │                               │                           │
    │ (с другого устройства)        │                           │
    │── открываю FORGE ────────────>│                           │
    │                               │── load state ────────────>│
    │<─ тот же прогресс! ───────────│                           │
```

Все данные хранятся на сервере Railway привязанные к твоему **Telegram ID**.
Зайди с телефона или компьютера — всё синхронизировано.

## Если что-то пошло не так

- **Бот не отвечает** → проверь `BOT_TOKEN` в переменных Railway
- **WebApp не открывается** → проверь `WEBAPP_URL` — должен быть полный URL с https://
- **Данные не сохраняются** → проверь что `DB_PATH=/data/forge.db` и Volume подключён в Railway
- **Ошибка при деплое** → зайди в Railway → Deployments → посмотри логи
